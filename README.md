# Nina Carducci Photographe professionnelle à Bordeaux

Je suis Jérémy Darbellay, développeur front-end freelance recruté par Kasa. 

Dans ce projet, nous allons optimiser le site d'une photographe professionnelle pour le référencement sur les moteurs de recherches. De plus, nous travaillerons quelques peu l'accessibilité et corrigerons deux petits bogues.

Ce projet est un site statique hébergé sur GHPages, aucune installation, ni back-end n'est requis.

# Démonstration

Ce projet étant l'optimisation d'un site statique, aucune installation n'est requise, ainsi, vous pouvez visiter le lien de démonstration et regarder les fichiers du répertoire. Voici un lien pour la version d'avant : [Site original de nina Carducci](https://ninacarducci.github.io/).
Vous retrouverez la démonstration du site à cette adresse : [démonstration](https://jeremydarbellay.github.io/projet_6_openclassrooms). 
Ce site est publié directement à partir de la branche main.

# Technologies utilisées

-----
![technologies utilisées](./assets/technos.png)

Pour ce projet nous avons travaillé avec [Jquery](https://jquery.com/)  et [Bootstrap](https://getbootstrap.com/), sans ajouter d'autres librairies, en utilisant les langages HTML5, CSS3 et JavaScript.

Le gros du travail a été l'intégration des optimisations pour moteur de recherche, ainsi plusieurs rapport [LightHouse](https://developer.chrome.com/docs/lighthouse/overview/) et [GTMetrix](https://gtmetrix.com/) ont été produits. De plus nous avons vérifié les principes de base de l'accessibilité via [Axe](https://www.deque.com/axe/devtools/) et [WAVE](https://wave.webaim.org/extension/).
